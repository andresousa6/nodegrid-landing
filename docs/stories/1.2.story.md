## Contexto & Referência

Site de referência: `https://design.aioxsquad.ai/`

**Padrões a adaptar para NodeGrid:**
- Marquee ticker no topo com CSS puro
- Secções numeradas em bracket notation: `[00]PORTFOLIO_` + `// subtitle`
- Hero com tipografia display MASSIVA (clamp 60px → 160px)
- Grid assimétrico featured: 1 card grande (2/3 width) + 2 pequenos empilhados (1/3)
- Filtros interactivos por categoria e stack (JavaScript puro, sem framework)
- Cards com número de projecto (`01`, `02`...), URL/domínio, categoria em caps mono
- Secções `[02]POR_CATEGORIA_` e `[03]POR_STACK_` com browsing temático

**Design system:** Manter tokens NodeGrid (`css/tokens.css`) — fundo `#070e07`, verde `#72b872`, gold `#f0d474`. A estética é dark-brutalist adaptada à paleta NodeGrid.

## Dev Notes

- **Substituição total:** `portfolio.html` é reescrito do zero. A Story 1.1 criou a estrutura base — lê-a como referência para a nav/footer mas não reutilizes o código de cards ou hero.
- **Ordem de implementação sugerida:** T3 (ticker) → T4 (hero) → T5+T6 (grid + cards) → T7 (filtros JS) → T8+T9 (secções categoria/stack) → T10+T11 (CTA + responsividade) → T12 (QA consola)
- **JavaScript:** Mantém num `<script>` no final do body. Máx. 80 linhas. Sem imports externos.
- **Ticker duplicação:** Para o marquee funcionar correctamente sem gap no loop, o conteúdo deve estar duplicado no HTML: `<span>NODEGRID · ... ·</span><span>NODEGRID · ... ·</span>` e o `translateX(-50%)` move exactamente metade.
- **Grid assimétrico:** O card `[01]` é `.card--featured` com `grid-row: 1 / 3`. Os cards `[02]` e `[03]` ocupam as 2 linhas da direita. O card `[04]` fica abaixo em full-width ou 2ª linha.
- **Tokens em falta:** Definir `:root { --text-secondary: var(--neutral-400); --text-tertiary: var(--neutral-600); --accent-subtle: var(--green-700); }` no topo do `<style>` (idem Story 1.1).

## Notas técnicas

- **Substituir** `portfolio.html` directamente (não criar novo ficheiro)
- **Preservar** o sistema de fallback de imagens (.png → .svg → inline)
- **Marquee CSS pattern:**
  ```css
  @keyframes marquee { from { transform: translateX(0) } to { transform: translateX(-50%) } }
  .ticker-inner { animation: marquee 24s linear infinite; white-space: nowrap; }
  ```
- **Grid assimétrico pattern:**
  ```css
  .featured-grid { display: grid; grid-template-columns: 3fr 2fr; grid-template-rows: auto auto; gap: 12px; }
  .card-featured { grid-row: 1 / 3; } /* ocupa as 2 linhas */
  ```
- **Filtro JS pattern:**
  ```javascript
  pills.forEach(pill => pill.addEventListener('click', () => {
    const filter = pill.dataset.filter;
    cards.forEach(card => {
      const match = filter === 'all' || card.dataset.category === filter || card.dataset.stack.includes(filter);
      card.style.opacity = match ? '1' : '0.2';
      card.style.pointerEvents = match ? 'auto' : 'none';
    });
  }));
  ```
- **Domínios fictícios** mas realistas (formato: `projecto.nodegrid.dev`) — servem apenas como credibilidade visual
